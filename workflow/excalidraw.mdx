The team uses Excalidraw as a shared visual whiteboard for planning, architecture sketches, and protocol diagrams. It requires no account and is accessible to all members via a shared link.

---

## What It Is Used For

- Diagramming the Jira task workflow state machine
- Sketching system architecture and service interaction diagrams
- Whiteboarding the cryptographic protocol flow during team meetings
- Planning database schema relationships
